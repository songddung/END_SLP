## 📅 1주차

### 목표

- 알고리즘과 자료구조 이해
- 핵심 템플릿 암기 및 3분 내 구현 가능
- 문제 풀이 기록 습관화

### 학습 계획

| 요일   | 학습 내용                                 |
| ------ | ----------------------------------------- |
| 월, 화 | DFS/BFS, Union-find, PriorityQueue 사용법 |
| 수, 목 | 시뮬레이션                                |
| 금     | 그래프 알고리즘, 세그먼트 트리            |
| 토     | 다익스트라                                |
| 일     | B형 기출 2문제                            |

### 템플릿별 목표 시간

| 템플릿              | 목표 시간         |
| ------------------- | ----------------- |
| BFS                 | 3분 내 구현 가능  |
| DFS                 | 2분 내 구현 가능  |
| 다익스트라          | 5분 내 구현 가능  |
| 유니온파인드        | 3분 내 구현 가능  |
| 방향벡터 + 범위체크 | 1분 내 구현 가능  |
| 세그먼트 트리 기본  | 10분 내 구현 가능 |
